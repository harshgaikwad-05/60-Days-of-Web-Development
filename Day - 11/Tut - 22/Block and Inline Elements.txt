BLOCK ELEMENTS vs INLINE ELEMENTS
--------------------------------

Think of a webpage like a floor full of toys üß∏

====================
BLOCK ELEMENTS
====================

Block elements are BIG toys.

What they do:
- Start on a NEW LINE
- Take the FULL WIDTH of the page
- Push other elements above or below them

Like:
A big box on the floor üì¶
Nothing can sit next to it

Examples of block elements:
- div
- p
- h1, h2, h3, h4, h5, h6
- section
- article
- header
- footer
- ul
- ol
- li

====================
INLINE ELEMENTS
====================

Inline elements are SMALL toys.

What they do:
- Stay on the SAME LINE
- Take ONLY the space they need
- Sit next to other inline elements

Like:
Stickers in a row ‚≠ê‚≠ê‚≠ê

Examples of inline elements:
- span
- a
- strong
- em
- b
- i
- img
- label

====================
IMPORTANT DIFFERENCE
====================

BLOCK:
- New line ‚ùå sit next to others
- Full width
- Big and chunky

INLINE:
- Same line ‚úÖ
- Only needed width
- Small and light

====================
FUN FACT
====================

You can change behavior using CSS!

Example:

display: block;
display: inline;
display: inline-block;
