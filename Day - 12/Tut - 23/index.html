CSS Outline

Outline is a non-box-model visual layer drawn outside the border. It does NOT affect layout, element size, or document flow.

  • Used for focus states
  • Common in accessibility
  • Does not take up space

CSS Box Shadow

box-shadow creates layered depth by adding shadows around elements. Widely used in Material UI, Bootstrap, and modern design systems.
